# 指标体系管理技能

## 框架结构

四层金字塔:

```
L0 策略层:7 个资产(A 股/港股/短债/中长债/美股/美债/黄金)
L1 基本面层:中美两国经济
L2 传导链路层:7 条链路 × 29 个节点
L3 指标层:118 个指标
```

每个指标挂在一个或多个节点上。一个指标可以挂多个节点(如布伦特原油挂 4 个节点),每个节点里的打分方向可以不同。

---

## 议题 1 后的关键变化

### "主项/子项"原则

- **主项**:同节点内被直接赋予权重、参与加权计算的指标(如社融、新增贷款、M1、M2)
- **子项**:权重=0 但保留展示的指标(如信贷结构 4 分项、PMI 购进价格、美国总 CPI 等)

### 权重集中到传导链路 E 列

- 汇总 Sheet N 列全部=0(不再使用)
- 所有权重信息都在**传导链路 Sheet 的 E 列公式**里
- 每个节点权重总和必须=1.0

## 新增指标的评估框架

当用户提出要加一个指标时,按以下 4 个问题评估:

**Q1:这个指标回答什么问题?**
- 如果和已有主项回答同一问题,应替换/归为子项,而非新增另一个主项
- 如果回答新问题(如"信贷是居民还是企业偏弱"),优先级高

**Q2:它挂哪个节点?**
- 一个指标只应挂和它因果关系最直接的节点
- 避免"相关性多=挂多节点"

**Q3:Wind 能拉到吗?**
- 能拉到 → 加到 Excel 汇总 Sheet
- 不能拉但有公开数据 → AI 搜索补充
- 完全无数据 → 放弃或等

**Q4:它是主项还是子项?**
- 如果它能独立反映一个维度(如盈利预期),当主项
- 如果它是已有主项的分解(如信贷结构分项之于新增贷款),当子项(权重=0,仅展示)
- 一个节点最好不超过 8-10 个主项

## 打分阈值设计原则

### 基于"对资产的含义"而非"指标本身好不好"

**错误**:PMI > 50 就打正分(因为 50 是荣枯线)

**正确**:PMI > 51 打 2 分、> 50 打 1 分、> 49.5 打 0 分、> 48 打 −1 分(因为对 A 股的影响是渐进的)

### 嵌入"预期差"维度

对有预期值(万得一致预测)的指标,优先看**实际值 vs 预期值的差距**,而非绝对值:
```excel
=IF(M12="",
  [绝对值打分],
  [预期差打分: H12-M12 >= 0.5 → 2 分, >= 0.1 → 1 分, ...]
)
```

### 加入"变化方向"加分

在绝对值基础上,叠加变化方向的加减分(±0.5):
```excel
+IF(L23<-0.05, 0.5, IF(L23>0.05, -0.5, 0))
```

## 重要的分析逻辑备忘

### 新增人民币贷款 = 居民 + 企业 + 非银

总量数据不能直接归因某个部门。现在有了居民短贷、居民中长贷、企业中长贷,可以做结构拆分(但议题 1 后它们权重=0,仅作分析辅助)。

### 企业中长贷 + 票据融资 的组合判断

- 企业中长贷↑ + 票据↓ = 真实信贷需求改善
- 企业中长贷↓ + 票据↑ = 银行用票据冲量,需求弱
- 两者同↑ = 银行加力投放
- 两者同↓ = 信用全面收缩

### PPI 转正需区分来源

- PMI 原材料购进价格 > 52 + 布伦特 > 100 = **成本推动型** PPI 转正(对企业利润利空)
- PMI 原材料购进价格 < 50 + PPI 转正 = **需求改善型**(利好)

### 黄金定价因子的当前权重(2026 年更新)

主流机构(JPM / 世界黄金协会)已从"实际利率 + 美元"的传统模型,扩展为:

1. **央行结构性购金**(最重要,17 年连续净买入)
2. **对美国财政赤字与美联储独立性的担忧**
3. **去美元化**
4. 美元 + 实际利率(传统因子,仍有贡献但非主导)
5. 投资需求(ETF + 实物金条)

**金油比概念在现代黄金分析中已边缘化**,不作为核心判断,仅作参考。
