# 宏观分析撰写技能(v2.2)

## Prompt 版本

当前版本:**v2.2**,完整 Prompt 文件由用户本地维护(`analysis_prompt_v2.2.md`)。

本文档记录**写作规则和硬规则**,供 Agent 改 Prompt 时参考。

---

## 排版样式(硬性要求)

### 信号流格式

必须带节点名称:
```
央行⛅️常规操作 → 银行间☀️☀️显著宽松 → 实体🌧️信用收缩 → 经济⛅️企稳观望 → 市场⛅️☀️温和活跃
```

不能只写 emoji + 标签(如 `⛅️ → ☀️☀️ → 🌧️`)。

### 段落结构

- 每段必须有 `#### 小标题`
- 小标题简短有力,如 `#### ⚠️ 核心断点:银行间 → 实体传导`
- 不能写大段不分行的文字

### 关键数据加粗

所有被引用的具体数值用 `**加粗**`,如"DR007 本月以来累计下行约 **10bp** 至 **1.32%**"。每段至少 2-3 个加粗数据。

### 段内分行

较长段落用 `**小标签:**` 分行:
```
**银行间端:** 资金充裕...
**实体端:** 信用扩张放缓...
**断点含义:** ...
```

## 分析结构(以断点为主线)

```
信号流(带节点名)

全景判断(2-3 句,不超 100 字)

---

#### 断点/堵点展开(如有,每个单独一段)
#### 其余节点补充(每个一段,正常的可合并)
#### 后续关注(1-2 句收尾)
```

**核心原则:不按节点逐个念数据,而是找到链路中最关键的矛盾点,围绕它展开。**

## 写作硬规则

### R1 数据边界

- 只能基于 JSON 中存在的字段做判断
- 总量数据不能推导分项归因(如"新增贷款偏弱"不能推导为"企业融资意愿不足",除非有企业中长贷数据)
- 判断必须有加粗数据锚定

### R2 日度/周度数据表述

- 日度数据优先用"本月以来"变动(`mtd_change` 字段),不用逐日变化
- **南向资金等波动大的指标,不能用单日数据做趋势判断**,必须用累计值
- 周度数据说"本周/上周",不说"上期"
- 模板:"DR007 本月以来累计下行约 **10bp** 至 **1.32%**"

### R3 语气规范

禁止:极度、暴涨、崩塌、恶化、一定、必然、说明了、证明了

替代:处于较低水平、录得较大幅度上行、表现偏弱、或反映、可能指向

因果推理只用"或反映""可能指向",不用断言。

### R4 术语通俗化

首次出现必须括号解释:
- "DR007(银行间 7 天回购利率)"
- "M2-M1 剪刀差(衡量资金活化程度的指标)"
- "R007 与 DR007 利差仅 4bp,说明不仅银行之间资金宽裕,非银机构(券商、基金等)的资金面同样宽裕"

**通胀保值国债的中文名优先于 TIPS**(2026-04-19 新增 R2.6):
- 用"10 年期通胀保值国债收益率",不用"TIPS 10Y"
- 首次出现时加解释:"通胀保值国债的收益率扣除通胀补偿后 = 实际利率"

利差类指标的通俗表述:
- "期限利差走阔" → "不同期限之间的收益率差距在扩大"
- "信用利差收窄" → "企业债相对国债的额外收益率在缩小,反映市场对信用风险担忧降低"

### R5 价格指标溯因规则

PMI/PPI 等价格相关指标上涨时,AI 必须检查油价是否异动。如果当前布伦特原油 > 100 美元:
- PMI 回升必须标注"需关注价格分项是否受能源成本推动"
- PPI 转正必须标注"或部分受油价推升,需区分成本推动型 vs 需求改善型"

### R6 篇幅控制

- 全景判断:不超 100 字
- 每个小标题段落:不超 150 字
- 全文:600-800 字(不含标题和信号流)

### R7 数值比较硬规则(2026-04-19 新增)

分析里出现"A 高于 B"、"A 低于 B"、"倒挂"、"大于/小于"等数值比较表述时:

- **必须在稿子里显式写出两边的数值**,或做快速算术验证
- 避免"想说什么就写什么"造成事实错误
- 项目历史曾出现过"1.475% 低于 1.18%"这种事故,教训深刻
- "倒挂"用法要精确:专指短端利率高于长端(或本应低的标的)

### R8 债市分析的"双轨"视角(2026-04-19 新增)

油价回落后,不能简单用"Bull Steepening 降息预期交易"单一解释债市变化。搜索结果显示,市场常表现为"Relief Rally 但保留对冲":

- 风险资产与避险资产**同时上涨**
- 收益率下行是**多层逻辑叠加**:期限溢价下降 + 远期降息预期 + 避险对冲需求
- 描述时要体现这种多层次,而非单因果

### R9 黄金分析的"主流框架"(2026-04-19 新增)

主流机构(JPM / 世界黄金协会)已从"实际利率 + 美元"的传统模型,扩展为新框架:

1. **央行结构性购金**(核心支撑)
2. **对美国财政赤字 / 美联储独立性的担忧**
3. **去美元化**
4. 美元 + 实际利率(传统因子,仍有贡献但非主导)
5. 投资需求(ETF / 实物)

**金油比在现代黄金分析中已边缘化**,不作为核心判断,仅作参考视角。

## AI 搜索补充规则

以下数据 Wind 无法提供,分析时需要 AI 调用搜索工具补充:

### 港股/美股盈利预期

- Wind 对港股(恒指/恒生科技)和美股(标普 500)的盈利一致预期覆盖不全
- 分析港股或美股链路时,调用搜索补充 Bloomberg / FactSet 的最新盈利增速数据
- 搜索关键词示例:"恒生指数 2026 Bloomberg 一致预期 EPS 增速"或 "S&P 500 earnings FactSet Q1 2026"
- 引用时必须标注来源和数据日期

### CME 降息概率

- 分析美股/美债/黄金链路时,搜索"CME FedWatch [当月]"获取降息概率
- 这个数据反映市场对美联储路径的定价
- 搜索关键词:"CME FedWatch 2026 年 4 月 降息概率"
- 如果搜不到最近 7 天内的数据,标注"最新 CME 降息概率数据未获取"

### 美联储官员讲话

- 分析美股/美债时,可搜索最近一周内的美联储官员公开讲话
- 搜索关键词:"美联储 [当月] 讲话 通胀 利率"

### TGA 异常变动解读(2026-04-19 经验)

- 美国 TGA 账户余额有周度/月度的季节性波动
- 4 月中(税日)/ 9 月中 / 12 月中 通常跳涨(税款入账)
- 分析异常变动时,搜索"TGA tax season [月份] liquidity"区分**季节性** vs **结构性**
- 例:2026-04-15 TGA 单周 +2,274 亿美元至 9,244 亿美元,彭博明确定性为税季效应

### 黄金定价新框架(2026-04-19 经验)

- 当黄金上涨与美元/利率信号不完全匹配时,查主流机构最新定价框架
- 搜索关键词:"gold rally central banks de-dollarization 2026"
- 关注 JPM / 世界黄金协会 / RBC / Citi 等机构的最新展望

### 搜索硬规则

- 搜索前检查 JSON 中 `meta.export_date` 字段,确认数据日期
- 严禁引用过期数据作为"当前"市场定价
- 搜索到的数据如果和 JSON 里的数据矛盾,**以 JSON 为准**(JSON 是一手数据),搜索结果作为补充参考

## 外部环境前置段

v11 新增了布伦特原油、PMI 购进价格等外部冲击指标。分析任何链路前,AI 应该先检查这些指标是否异动:

- 布伦特原油 > 100 美元 → 必须在全景判断中提及外部环境
- VIX > 25 → 全球风险偏好承压
- 美国 CPI 同比 > 3.5% → 通胀风险上升
- 金油比 < 25 → 参考视角:市场处于强通胀交易模式(当前已降级为参考)

## HTML 展示规范(2026-04-19 新增)

### Sparkline 趋势图

- 每条链路嵌入 3-6 个 sparkline 图,紧贴对应论点
- 月度数据用 12 月,日度用 60 日
- 衍生指标(利差/比率)可实时计算(从原始指标的 history 算)
- 双轴图:两个量纲不同的指标同图时使用(如 PMI 50 附近 + 油价 90+)

### 默认展开策略

- 重点链路(通常 A 股 + 美股)默认展开
- 其他链路折叠,点击展开

### 视觉风格

- 暗色投研终端风(`#0d1117` 底色 + 琥珀色 `#f0b90b` 强调)
- 无图表库依赖,纯 SVG 手写
- 单文件 HTML,可离线分发

## 分析测试 Checklist

每次改完 Prompt 或指标后,用以下清单验证分析质量:

- [ ] 信号流是否带节点名
- [ ] 全景判断是否在 100 字以内
- [ ] 是否有断点分析(如果数据显示有断点)
- [ ] 关键数据是否加粗
- [ ] 日度数据是否用了 `mtd_change` 而非逐日变化
- [ ] 南向资金是否用累计值而非单日
- [ ] 术语是否首次出现时解释了
- [ ] **TIPS 是否改用"通胀保值国债"**
- [ ] **数值比较是否两边数值都写出或验证过**
- [ ] PPI/PMI 是否检查了油价溯因
- [ ] **债市描述是否考虑了"Relief Rally 但保留对冲"的多层次**
- [ ] **黄金描述是否用了新框架(央行购金+去美元化+独立性担忧)而非旧框架**
- [ ] 是否有投资建议或仓位建议(不应该有)
- [ ] 全文是否在 800 字以内
