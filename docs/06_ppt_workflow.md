# PPT 制作工作流（可选展示分支）

> 本文档记录宏观投研自动化工程的 PPT 展示规范。PPT 不是当前主链路的必选交付，而是 HTML 展示之外的可选投研汇报形态。

---

## 定位

PPT 的目标不是把 Excel 数据机械搬到页面上，而是把宏观数据整理成**投研汇报页**：

1. 先确认数据边界和最新值
2. 再形成页面主判断
3. 最后选择能支撑判断的图表

一句话原则：

> 图表服务观点，不为了“看起来专业”而堆图。

---

## 输入与输出

### 输入

| 输入 | 用途 |
|------|------|
| `macro_final_v11.2.xlsx` | 当前核心 Excel 样例，用于读取指标最新值与历史月度数据 |

| 用户给定展示样例 | 用于确定版式、颜色、信息密度和汇报风格 |

### 输出

| 输出 | 是否入库 | 说明 |
|------|---------|------|
| PPTX 成品 | 默认不入库 | 每轮生成物，放本地 `outputs/` |
| 预览 PNG | 默认不入库 | 仅用于检查页面是否清晰 |
| PPT 规范文档 | 入库 | 本文档、skill、模板说明 |
| 自动化脚本 | 后续视情况入库 | 只有稳定复用后再放入 `scripts/` |

## 设计语言

> PPT 设计风格先按 `docs/07_presentation_system.md` 判断。本文件默认描述"宏观投研风格";如果页面是工作汇报、项目推进或机制建设,应改用 `templates/ppt/juncheng_work_report_style.md`。

### 整体风格

采用机构投研报告风格，而不是营销型路演风格：

- 白底为主
- 深蓝标题
- 红色强调核心判断
- 表格和图表可以密集，但必须可读
- 不使用花哨渐变、装饰背景、水印、贴纸式元素
- 每页只回答一个主问题

### 推荐颜色

| 用途 | 颜色 |
|------|------|
| 主标题深蓝 | `#102641` |
| 强调红 | `#C00024` |
| 图表标题蓝 | `#155D8B` |
| 正文黑 | `#202020` |
| 辅助灰 | `#5F6670` |
| 表格边框 | `#D9DEE7` |

### 版式规格

| 项目 | 建议 |
|------|------|

## 页面类型

### 1. 总览页

用途：回答“这条链路当前整体状态是什么？”

推荐结构：

```text
标题
三条核心判断
月度热力表
脚注
```

适合图表：

- 月度热力表
- 分类表格
- 指标矩阵

示例：A 股经济基本面页。

分类逻辑：

| 分类 | 典型指标 |
|------|---------|
| 领先指标 | 制造业 PMI、非制造业 PMI、PMI 原材料购进价格 |
| 同步指标 | 工业增加值、社零、固定资产投资及分项、商品房销售面积、出口 |
| 滞后指标 | CPI、PPI |

注意：同一页只讲一个传导维度。A 股经济基本面页不混入社融、M1、M2，后者应放在信用/流动性页。

### 2. 证据拆解页

用途：回答“为什么得出这个判断？”

推荐结构：

```text
标题
三条判断
左：景气或生产证据
中：价格或成本证据
右：需求或约束证据
底部核心判断框
```

示例：PPI 转正偏输入、内需修复仍需观察。

### 3. 价格传导页

用途：解释 CPI、PPI、PMI 原材料价格、油价等价格指标之间的关系。

关键规则：

- PPI 转正不能直接写成内需恢复
- 如果 PMI 原材料购进价格明显上行，需要提示“或更多来自上游原材料输入”
- CPI 未同步上行时，应避免把价格改善写成居民端需求确认
- 如涉及油价，必须按项目硬规则检查能源价格影响

### 4. 流动性页

用途：单独展示社融、M1、M2、银行间资金和信用派生。

注意：不要把流动性指标混进经济基本面热力表，除非页面主题就是“实体信用传导”。

## 图表选择规则

### 通用原则

每张图制作前先回答：

> 这张图希望读者一眼看出什么？

如果回答不清楚，优先改成表格或删掉。

### 热力表

适用场景：

- 指标多
- 需要展示月度横截面
- 需要看全景强弱

规则：

- 颜色按每一行自身历史区间做相对强弱，不跨指标直接比较颜色深浅
- 表头必须清楚标出季度和月份
- 指标要分组，避免读者不知道先看什么

### PMI 折线图

适用场景：

- 展示制造业 PMI、生产、新订单、新出口订单之间的差异

规则：

- 必须标出 50 荣枯线
- 纵轴不要默认从 0 开始，建议固定在 `44-52` 或按数据窄区间调整
- 如果新出口订单低于 50，需要在文字中显式说明

### 价格类折线图

适用场景：

- 展示 PPI、CPI、PMI 原材料购进价格、油价等

规则：

- 不同量级不要硬放同一坐标轴
- PPI/CPI 可以放一张图
- PMI 原材料购进价格建议单独成图或使用副轴
- 需要用文字解释“输入性价格”和“内需价格”的区别

### 内需约束图

适用场景：

- 社零、固投、地产销售、民间投资等指标对比

规则：

- 如果指标正负差异很大，横条图容易让小幅变化看不清
- 优先用 `上期 → 本期` 对比表
- 必须同时呈现当前水平和边际变化

## 文案规则

PPT 文案要沿用项目分析规则：

- 不使用“证明了、必然、一定”等绝对化措辞
- 使用“或反映、可能指向、仍需观察”等表述
- 出现数值比较时写出两边数值
- 价格类指标上涨必须溯因
- 不超出 Excel/JSON 已有数据做归因

示例：

```text
PPI 由 -0.9% 转为 0.5%，同时 PMI 原材料购进价格从 54.8 升至 63.9，
或更多来自上游原材料输入，不能简单等同于内需已经走强。
```

## 本轮 A 股经济页经验

本轮制作形成了几个固定经验：

1. 第一页适合用“热力表 + 三条判断”做全景页
2. 第二页适合拆成“PMI、价格、内需”三块证据
3. PMI 折线图必须处理纵轴，否则线会挤在一起
4. PPI/CPI 和 PMI 原材料价格不宜硬放同一轴
5. 社零、固投、地产销售、民间投资更适合做上期到本期的对比表
6. A 股经济基本面页不放社融、M1、M2，避免混入信用传导主题

## 后续自动化方向

如果未来决定把 PPT 流程自动化，可以按以下顺序开发：

1. Excel/JSON 数据抽取脚本
2. 热力表生成脚本
3. PPT 模板构建脚本
4. 渲染预览检查
5. 多资产页面模板扩展

当前阶段先保留为**半自动规范**：Agent 按规范制作，用户确认后再考虑脚本化。
